# Maximum Subarray Sum – Kadane’s Algorithm

## Problem Statement
Given an integer array, find the contiguous subarray with the largest sum.

## Approach
- Use Kadane’s Algorithm
- Maintain current_sum
- Reset if negative
- Track max_sum

## Time: O(n)
## Space: O(1)

Example:
Input: [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
